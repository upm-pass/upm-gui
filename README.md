## **Installation**
Installation guide for upm (uwu password manager)

### **Linux**

```bash
$ git clone https://github.com/upm-pass/upm-gui.git
```

cd project directory 
```bash 
cd upm-gui/
```

to install upm-gui run
```bash
./install.sh
```
After that run gupm from your terminal or application launcher
